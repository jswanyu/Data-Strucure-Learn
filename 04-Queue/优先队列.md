# 优先队列

普通队列：先进先出，后进后出

优先队列：出队顺序和入队顺序无关，和优先级相关



优先队列也实现队列的接口



优先队列实现方式及其时间复杂度：

|              | 入队    | 出队（取最大元素） |
| ------------ | ------- | ------------------ |
| 普通线性结构 | O(1)    | O(n)               |
| 顺序线性结构 | O(n)    | O(1)               |
| 堆           | O(logn) | O(logn)            |
